# 🎬 Movie Recommendation System

A powerful **Content-Based Movie Recommendation System** that suggests movies similar to your preferences using advanced Natural Language Processing (NLP) techniques.

This project leverages **TF-IDF Vectorization** and **Cosine Similarity** to analyze movie descriptions and deliver highly relevant recommendations.

## 🧠 Project Overview

This system recommends movies by analyzing textual data such as **movie overviews, genres, and metadata**.

Instead of relying on user ratings (collaborative filtering), it uses a **content-based approach**, ensuring recommendations even for new users (cold start problem solved).

---

## ✨ Key Features

* 🔍 Intelligent **content-based filtering**
* 🧠 NLP-powered similarity using **TF-IDF**
* 📊 Uses **Cosine Similarity** for accurate matching
* 🎬 Returns **Top 5 highly relevant movie recommendations**
* ⚡ Fast and lightweight (runs in seconds)
* ☁️ Fully executable on **Google Colab**
* 🌐 Interactive UI using **Streamlit**

## ⚙️ How It Works

1. 📥 Load movie dataset (TMDB 5000)
2. 🧹 Preprocess text data (cleaning & feature selection)
3. 🔤 Convert text → numerical vectors using **TF-IDF**
4. 📐 Compute similarity using **Cosine Similarity**
5. 🎯 Recommend top 5 similar movies
